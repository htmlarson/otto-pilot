# Otto Pilot

A Chrome extension allowing for inline play of Floatplane content from within YouTube

## A word of warning

The developer of this extension has abandoned this project, and as such, **you are on your own.**. It is also designed for Chrome, [so use at your own risk.](https://arstechnica.com/information-technology/2019/09/no-it-wasnt-a-virus-it-was-chrome-that-stopped-macs-from-booting/).
